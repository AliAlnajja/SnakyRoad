PROJECT TITLE: Snaky Road
PURPOSE OF PROJECT: Game programming final project
VERSION or DATE: 2020-12-09 version 1
HOW TO START THIS PROJECT: Click on run, then the space bar to start game
AUTHORS: Ali, Carla, Christopher and Igor
USER INSTRUCTIONS:
You play as a snake and you have to try to reach the eggnest while avoiding the many enemies
There are 3 levels in totals with different enemies on each one
There is an instruction screen at the beginning of the game that shows the controls and objectives
